This repository hosts a desktop application developed using **C#** and the **Windows Forms (WinForms)** framework within the **.NET** environment. The focus is on delivering an intuitive User Interface (UI) that mimics a standard desktop calculator experience.

***

## üí° Key Features & Functionality

The application is designed to handle the following core mathematical operations and features:

| Feature Category | Description |
| :--- | :--- |
| **Basic Arithmetic** | Supports addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`). |
| **Memory Functions** | Basic memory storage (MR, MS, M+, M-). |
| **Input Handling** | Clear entry (`CE`), clear all (`C`), and backspace functionality. |
| **Display** | Clear, readable display for input and results. |
| **(Optional) Advanced** | Potential for percentage (%), square root, and reciprocal functions. |

## üöÄ Getting Started

Follow these steps to set up the project locally and run the Calculator application:

1.  **Clone the Repository:**
    ```bash
    git clone [Your-Repository-URL]
    ```

2.  **Open the Solution:**
    * Navigate to the cloned repository folder.
    * Double-click the **`calculator.sln`** file to open the solution in Visual Studio.

3.  **Build the Project:**
    * In Visual Studio, go to the **Build** menu and select **Build Solution** (or press `Ctrl + Shift + B`).

4.  **Run the Application:**
    * Press the green **Start** button (or `F5`) to execute the application in Debug mode. The calculator window will appear, ready for use.
